# Система онлайн-голосования

Простая и надежная система для проведения выборов.

## Быстрый старт (через GitHub)

1.  **Клонируйте репозиторий** на ваш сервер:
    ```bash
    git clone https://github.com/ВАШ_ЮЗЕРНЕЙМ/voting.git
    cd voting
    ```

2.  **Запустите через Docker Compose**:
    ```bash
    docker-compose up -d --build
    ```

3.  **Готово!**
    *   Сайт: `http://ваш-ip`
    *   Админка: `http://ваш-ip/admin`
    *   Код администратора: `Kazan1552`

    *Примечание: База данных уже настроена и содержит тестовых кандидатов и коды.*

## Устранение неполадок (Ошибка 500)

Если при входе возникает ошибка "Internal Server Error", выполните полную переустановку базы данных:

```bash
docker-compose exec voting-app python scripts/force_reset_db.py
```

## Коды доступа
Коды для избирателей находятся в файле `codes.csv`. Вы можете сгенерировать новые, заменив список в `list1.csv` и запустив скрипт генерации (или просто отредактировав `codes.csv` и перезапустив `setup_full.py`).
