# Инструкция по развертыванию системы голосования

## Вариант 1: Docker (Рекомендуемый)

Этот способ самый быстрый и надежный. Требуется установленный Docker и Docker Compose.

1.  **Скопируйте файлы проекта** на сервер.
2.  **Настройте переменные окружения** в файле `docker-compose.yml` (по желанию):
    *   `SECRET_KEY`: Секретный ключ (замените на случайную строку).
    *   `ADMIN_CODE`: Код администратора (по умолчанию `admin123`).
3.  **Запустите контейнер**:
    ```bash
    docker-compose up -d --build
    ```
4.  **Инициализируйте базу данных**:
    ```bash
    docker-compose exec voting-app python scripts/setup_full.py
    ```
    *Это создаст базу данных, добавит кандидатов и импортирует коды из `codes.csv`.*

5.  **Готово!** Сайт доступен по адресу `http://ваш-ip-адрес`.

## Вариант 2: Ручная установка (Linux/Ubuntu)

Если вы не используете Docker.

1.  **Установите Python и venv**:
    ```bash
    sudo apt update
    sudo apt install python3 python3-venv python3-pip
    ```

2.  **Клонируйте/скопируйте проект** в папку `/opt/voting`.

3.  **Создайте виртуальное окружение**:
    ```bash
    cd /opt/voting
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

4.  **Инициализируйте базу данных**:
    ```bash
    python scripts/setup_full.py
    ```

5.  **Запустите сервер (Gunicorn)**:
    ```bash
    gunicorn -w 4 -b 0.0.0.0:80 app:app
    ```
    *(Для работы на 80 порту может потребоваться sudo или настройка прав)*

---

## Управление голосованием

1.  Перейдите по адресу `/admin`.
2.  Введите код администратора (из `docker-compose.yml` или `.env`).
3.  Нажмите **"Запустить голосование"**, когда будете готовы.
4.  Раздайте коды из файла `codes.csv` избирателям.
